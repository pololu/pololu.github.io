Detailed Description

Controls motor speed and direction on the 3pi+ 32U4.

This library uses Timer 1, so it will conflict with any other libraries using that timer.

void Pololu3piPlus32U4::Motors::flipLeftMotor ( bool  flip)
static

Flips the direction of the left motor.

You can call this function with an argument of true if the left motor of your 3pi+ was not wired in the standard way and you want a positive speed argument to correspond to forward movement.

Parameters
flipIf true, then positive motor speeds will correspond to the direction pin being high. If false, then positive motor speeds will correspond to the direction pin being low.
